# Python To-Do List App

A desktop To-Do List application built with Python and Tkinter.

## Features

- Add tasks
- Mark tasks as completed
- Remove tasks
- Task statistics
- Automatically saves tasks using JSON
- Simple and user-friendly interface

## Technologies Used

- Python
- Tkinter
- JSON

## Project Structure

```
ToDoList.py
tasks.json
README.md
```

## How to Run

1. Install Python 3
2. Download this repository
3. Run

```bash
python ToDoList.py
```
